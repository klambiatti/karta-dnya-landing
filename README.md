# «Карта дня» — лендинг (статический)

Одностраничник для Telegram‑бота с метафорическими ассоциативными картами.

## Быстрый старт

- Открой `index.html` в браузере (двойной клик)  
  или подними локально:

```bash
cd karta-dnya-landing
python3 -m http.server 8080
```

Потом открой `http://localhost:8080`.

## Плейсхолдеры ссылок

В `main.js` замените:

- `BOT_USERNAME` → например `@MAC_of_the_day_bot`
- `AUTHOR_USERNAME` → например `sergei_username`

Ссылки на кнопках (CTA) обновятся автоматически.

## Тексты Hero (ещё 2 варианта)

Вариант B:
- Заголовок: **Одна карта. Один вопрос. Пара минут тишины.**
- Подзаголовок: *Бот помогает мягко посмотреть на день через образ карты — без давления и без «правильно».*

Вариант C:
- Заголовок: **Тихий сад внутри дня — на 2 минуты**
- Подзаголовок: *Открываешь бота, тянешь карту и читаешь вопрос. Иногда этого достаточно, чтобы вернуться к себе.*

## Аналитика (опционально)

В `main.js` оставлено место под подключение Plausible/GA без ключей. Лучше добавлять после MVP.

## Деплой

Подойдёт любой статический хостинг:

- **Netlify**: просто перетащите папку или подключите репозиторий, publish directory = `karta-dnya-landing/`
- **Vercel**: как Static Site (output = `karta-dnya-landing/`)
- **GitHub Pages**: из папки / корня репозитория (если перенесёте проект в репо)
