# «Карта дня» — лендинг (статический)

Одностраничник для Telegram‑бота с метафорическими ассоциативными картами.

## Быстрый старт

- Открой `index.html` в браузере (двойной клик)  
  или подними локально:

```bash
cd karta-dnya-landing
python3 -m http.server 8080
```

Потом открой `http://localhost:8080`.

## Плейсхолдеры ссылок

В `main.js` замените:

- `BOT_USERNAME` → сейчас стоит `@mac_of_the_day_bot` (можно поменять при необходимости)
- `AUTHOR_USERNAME` → например `sergei_username`

Ссылки на кнопках (CTA) обновятся автоматически.

## Картинки-заглушки и что заменить (ТЗ)

Сейчас на сайте стоят лёгкие SVG‑заглушки в `assets/`:

- `assets/hero-card.svg` — картинка в мокапе карты на первом экране (соотношение **4:3**)
- `assets/og.svg` — картинка для превью при шаринге (OpenGraph), размер **1200×630**
- `assets/favicon.svg` — фавиконка (SVG)
- `assets/grain.svg` — «зерно/бумага» поверх фона (тайл)
- `assets/icons/*.svg` — иконки для карточек «Что внутри» (монохром)

Если будете заменять на реальные изображения, рекомендую подготовить:

- **Hero‑изображение карты**: `hero-card.webp` (или `.png`), **1200×900** (4:3), тёмный фон/бумага + тонкая «золото‑линия».  
  В `styles.css` заменить `background-image` у `.card-mock__art` на новый файл.
- **OpenGraph**: `og.png` или `og.webp`, **1200×630**, с крупным заголовком «Карта дня» и короткой строкой (или просто красивая композиция).  
  В `index.html` заменить `og:image`.
- **Фон‑текстура (опционально)**: `grain.png` 320×320 (тайл), очень лёгкая (прозрачность 5–10%), чтобы не «шумела».
- **Иконки (опционально)**: 5 шт., SVG монохром (белый/золото), 24×24 viewBox.

Важно:
- Лучше **WebP** для фото/рендеров (меньше вес), SVG — для линейных знаков.
- Держите контраст: золотой/белый на графите, без «ядовитых» оттенков.

## Тексты Hero (ещё 2 варианта)

Вариант B:
- Заголовок: **Одна карта. Один вопрос. Пара минут тишины.**
- Подзаголовок: *Бот помогает мягко посмотреть на день через образ карты — без давления и без «правильно».*

Вариант C:
- Заголовок: **Тихий сад внутри дня — на 2 минуты**
- Подзаголовок: *Открываешь бота, тянешь карту и читаешь вопрос. Иногда этого достаточно, чтобы вернуться к себе.*

## Аналитика (опционально)

В `main.js` оставлено место под подключение Plausible/GA без ключей. Лучше добавлять после MVP.

## Деплой

Подойдёт любой статический хостинг:

- **Netlify**: просто перетащите папку или подключите репозиторий, publish directory = `karta-dnya-landing/`
- **Vercel**: как Static Site (output = `karta-dnya-landing/`)
- **GitHub Pages**: из папки / корня репозитория (если перенесёте проект в репо)

## Иконки

Иконки в `assets/icons/*.svg` — **Feather Icons** (MIT, open-source). Если захотите — можем заменить на другой набор (например, Lucide) без изменений в верстке.
